# Architecture Decision Records (ADRs)

Cada decisão de arquitetura significativa vira um arquivo `ADR-XXXX-<slug>.md`.

## Formato

```
# ADR-XXXX — Título
- Status: Proposta | Aceita | Substituída por ADR-YYYY
- Data: AAAA-MM-DD
## Contexto
## Decisão
## Consequências
## Alternativas consideradas
```

## Índice
- ADR-0001 — Stack técnica
- ADR-0002 — Núcleo determinístico e separação core/render
- ADR-0003 — Separação arte × lógica (manifesto de assets)
- ADR-0004 — Monetização e entitlements
- ADR-0005 — Offline-first com Supabase posterior
