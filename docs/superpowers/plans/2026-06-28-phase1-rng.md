# RNG determinístico (Item 1.1) — Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Entregar um PRNG determinístico, portável e semeável em `src/core/rng/`, com API `next/range/int/pick/fork/clone` e suporte a replay via estado uint32.

**Architecture:** `mulberry32` (estado uint32, passo único) como motor; `xmur3` para hashear a seed (`string|number`) no estado inicial. Uma classe `Rng` mutável guarda o estado e inlina o passo (zero alocação no caminho quente); toda aleatoriedade deriva de uma única primitiva `nextUint32()`. `fork` deriva streams independentes do seed inicial.

**Tech Stack:** TypeScript estrito, Vitest. Sem dependências novas. `Math.imul` + `>>> 0` para aritmética uint32 portável.

## Global Constraints

- `src/core/` é headless e determinístico: **proibido** `Math.random`, `Date`, `Date.now`, `performance.now`, `setTimeout`, `setInterval`, `requestAnimationFrame`, `window`, `document`, `localStorage`, `fetch`, e imports de `phaser`/`preact`/`@preact/*`. (Guarda dupla: ESLint `src/core/**` + teste `tests/determinism/no-forbidden-apis...`.)
- TypeScript `strict: true`. Sem `any` sem justificativa comentada.
- Aritmética PRNG em uint32: usar `Math.imul(...)` e normalizar com `>>> 0` / `| 0`.
- Aliases de import: `@core/*` → `src/core/*`. Testes em `tests/` (Vitest, `*.test.ts`).
- Testes de **determinismo** ficam em `tests/determinism/` (entram em `npm run test:determinism` do CI). Testes de API/distribuição em `tests/core/rng/`.
- Sem alocação por chamada em `nextUint32`/`next` (caminho quente).

```ts
// Primitivas puras do PRNG mulberry32 e do hash de seed xmur3.
// Determinísticas e portáveis: aritmética uint32 via Math.imul e >>> 0.

/** Incremento do passo mulberry32 (constante do algoritmo). */
export const MULBERRY32_INCREMENT = 0x6d2b79f5;

/**
 * Mistura um estado (já avançado pelo incremento) no valor de saída uint32.
 * Esta é a única fonte de saída do RNG — `Rng` avança o estado e chama isto.
 */
export function scramble(state: number): number {
  let t = state;
  t = Math.imul(t ^ (t >>> 15), 1 | t);
  t = (t + Math.imul(t ^ (t >>> 7), 61 | t)) ^ t;
  return (t ^ (t >>> 14)) >>> 0;
}

/** Hash estável de string → uint32 (xmur3), usado para semear o estado inicial. */
export function xmur3Hash(input: string): number {
  let h = 1779033703 ^ input.length;
  for (let i = 0; i < input.length; i++) {
    h = Math.imul(h ^ input.charCodeAt(i), 3432918353);
    h = (h << 13) | (h >>> 19);
  }
  h = Math.imul(h ^ (h >>> 16), 2246822507);
  h = Math.imul(h ^ (h >>> 13), 3266489909);
  h ^= h >>> 16;
  return h >>> 0;
}
```

`src/core/rng/rng.ts`:
```ts
import { MULBERRY32_INCREMENT, scramble, xmur3Hash } from './mulberry32';

const UINT32 = 0x1_0000_0000; // 2^32

export interface Rng {
  readonly seed: string;
  readonly state: number;
  nextUint32(): number;
  next(): number;
  range(min: number, max: number): number;
  int(min: number, max: number): number;
  pick<T>(array: readonly T[]): T;
  fork(streamId: string | number): Rng;
  clone(): Rng;
}

/** Hash estável de uma seed (string|number) → estado inicial uint32. */
export function hashSeed(seed: string | number): number {
  return xmur3Hash(String(seed));
}

class Mulberry32Rng implements Rng {
  readonly seed: string;
  private _state: number;

  constructor(seed: string, state: number) {
    this.seed = seed;
    this._state = state | 0;
  }

  get state(): number {
    return this._state >>> 0;
  }

  nextUint32(): number {
    this._state = (this._state + MULBERRY32_INCREMENT) | 0;
    return scramble(this._state);
  }

  next(): number {
    return this.nextUint32() / UINT32;
  }

  range(_min: number, _max: number): number {
    throw new Error('not implemented'); // Task 3
  }

  int(_min: number, _max: number): number {
    throw new Error('not implemented'); // Task 3
  }

  pick<T>(_array: readonly T[]): T {
    throw new Error('not implemented'); // Task 3
  }

  fork(_streamId: string | number): Rng {
    throw new Error('not implemented'); // Task 4
  }

  clone(): Rng {
    return new Mulberry32Rng(this.seed, this._state);
  }
}

/** Cria um RNG a partir de uma seed estável. */
export function createRng(seed: string | number): Rng {
  const normalized = String(seed);
  return new Mulberry32Rng(normalized, hashSeed(normalized));
}

/** Reconstrói um RNG a partir de um estado uint32 salvo (replay/golden master). */
export function rngFromState(seed: string | number, state: number): Rng {
  return new Mulberry32Rng(String(seed), state | 0);
}

export { Mulberry32Rng };
```

```ts
  range(min: number, max: number): number {
    if (min >= max) return min;
    return min + this.next() * (max - min);
  }

  int(min: number, max: number): number {
    if (min >= max) return min;
    return min + Math.floor(this.next() * (max - min + 1));
  }

  pick<T>(array: readonly T[]): T {
    if (array.length === 0) throw new Error('pick: array vazio');
    return array[this.int(0, array.length - 1)];
```
