## Global Constraints (valem para TODAS as tasks)

- **REGRA 1 — não tocar `src/core/`.** Input/pausa vivem em `src/render/`. O core já faz a
  detecção de borda (`input.flap && !lastFlap`). Pausar = a `GameScene` deixa de chamar
  `loop.advance` (logo, `step`); nunca mutar o `WorldState` fora de `step`.
- **REGRA 3 — alocação-zero no caminho quente.** `sample()` é chamado 1×/step: deve **reusar**
  o mesmo objeto `InputFrame` (sem `new`/literal por chamada).
- **REGRA 4 — sem string de UI hardcoded.** O cue de pausa é **gráfico** (escurecimento), sem
  texto. Nenhuma chave i18n nova nesta fatia.
- **Semântica do flap:** a `InputSource` reporta o **estado atual** do botão (segurado/solto);
  a borda (um flap por pressão) é do core. Suportar **múltiplas fontes simultâneas** (dedo +
  tecla) e **autorepeat** via `Set<string>` de ids. Um **latch** garante que um tap sub-frame
  (press+release entre dois samples) ainda produza 1 flap.
- **Casca sem teste de unidade:** `controls.ts` (listeners DOM) e o gate/overlay da `GameScene`
  (Phaser) seguem o padrão de 2.1 (não testados por unidade; verificação visual). A **lógica
  testável** está em `FlapInputSource`/`PauseController` (puros, env node, sem DOM).
- TypeScript estrito, sem `any`. Testes Vitest (env node). `npm run check` e `npm test` verdes.


Adicionar:

```ts
/**
 * Fonte de flap dirigida por DOM (toque/clique/tecla), amostrada 1×/step.
 * Reporta o estado ATUAL do botão; a borda (1 flap por pressão) é do core.
 * - `active`: ids das fontes seguradas (multi-fonte + dedup de autorepeat).
 * - `latched`: garante 1 flap mesmo num tap sub-frame (press+release entre samples).
 * Alocação-zero: `sample()` reusa `this.frame`.
 */
export class FlapInputSource implements InputSource {
  private readonly active = new Set<string>();
  private latched = false;
  private readonly frame: InputFrame = { flap: false };

  /** Pressiona uma fonte (id estável: `pointer:<id>`, `key:Space`, …). */
  press(id: string): void {
    this.active.add(id);
    this.latched = true;
  }

  /** Solta uma fonte; soltar id ausente é no-op. */
  release(id: string): void {
    this.active.delete(id);
  }

  /** Zera held + latch (anti-flap-fantasma ao pausar). */
  reset(): void {
    this.active.clear();
    this.latched = false;
  }

  sample(): InputFrame {
    this.frame.flap = this.active.size > 0 || this.latched;
    this.latched = false;
    return this.frame;
  }
}

/**
 * Estado de pausa da camada de render. Pausar = a GameScene deixa de chamar step.
 * `onPause` dispara ao ENTRAR em pausa (a integração liga em FlapInputSource.reset).
 */
export class PauseController {
  private _paused = false;
  onPause?: () => void;

  get paused(): boolean {
    return this._paused;
  }

  pause(): void {
    if (this._paused) return;
    this._paused = true;
    this.onPause?.();
  }

  resume(): void {
    this._paused = false;
  }

  toggle(): void {
    if (this._paused) this.resume();
    else this.pause();
  }
}
```

### Testes (`tests/render/input.test.ts`) — escrever ANTES (TDD), todos devem falhar→passar

Manter o teste de `NullInputSource`. Adicionar:

- **inicial:** `new FlapInputSource().sample()` ⇒ `{flap:false}`.
- **press/release:** `press('a')` ⇒ `sample().flap===true`; depois `release('a')` ⇒
  próximo `sample().flap===false`.
- **held por vários samples:** `press('a')`; 3 `sample()` seguidos ⇒ `flap:true` em todos
  (a fonte NÃO faz edge; quem faz é o core).
- **multi-fonte:** `press('a'); press('b'); release('a')` ⇒ `sample().flap===true`;
  `release('b')` ⇒ `false`.
- **autorepeat/dedup:** `press('a'); press('a')` (mesmo id); um único `release('a')` ⇒
  `sample().flap===false`.
- **latch (tap sub-frame):** `press('a'); release('a')` **sem** sample entre eles ⇒ o
  próximo `sample().flap===true` e o **seguinte** `===false` (exatamente 1 flap).
- **alocação-zero (reuso):** `const s = new FlapInputSource(); expect(s.sample()).toBe(s.sample())`
  (mesma referência). Idem para `NullInputSource`.
- **reset/anti-fantasma:** `press('a'); s.reset();` ⇒ `sample().flap===false`.
- **PauseController:** inicial `paused===false`; `toggle()` alterna p/ true e de volta;
  `pause()` 2× é idempotente (continua true); `resume()` 2× idempotente; `onPause` é chamado
  exatamente 1× ao entrar em pausa (não em `pause()` repetido com já-pausado).

```ts
export const PAUSE_KEYS: readonly string[] = ['KeyP', 'Escape'];

/** Overlay de pausa: escurecimento sem texto (rótulo i18n é 2.4/2.6). */
```

### `controls.ts` (novo — casca DOM, sem phaser)

```ts
import { FLAP_KEYS, PAUSE_KEYS } from './constants';
import type { FlapInputSource, PauseController } from './input';

/**
 * Casca DOM: encaminha toque/clique/tecla para as peças puras (sem lógica de jogo).
 * Retorna um cleanup que remove os listeners.
 */
export function bindGameControls(
  target: Window,
  controls: { flap: FlapInputSource; pause: PauseController },
): () => void {
  const { flap, pause } = controls;
  const onPointerDown = (e: PointerEvent): void => flap.press(`pointer:${e.pointerId}`);
  const onPointerUp = (e: PointerEvent): void => flap.release(`pointer:${e.pointerId}`);
  const onKeyDown = (e: KeyboardEvent): void => {
    if (PAUSE_KEYS.includes(e.code)) {
      pause.toggle();
      return;
    }
    if (FLAP_KEYS.includes(e.code)) {
      e.preventDefault(); // Space/ArrowUp não rolam a página
      flap.press(`key:${e.code}`);
    }
  };
  const onKeyUp = (e: KeyboardEvent): void => {
    if (FLAP_KEYS.includes(e.code)) flap.release(`key:${e.code}`);
  };
  const onBlur = (): void => pause.pause();

  target.addEventListener('pointerdown', onPointerDown);
  target.addEventListener('pointerup', onPointerUp);
  target.addEventListener('pointercancel', onPointerUp);
  target.addEventListener('keydown', onKeyDown);
  target.addEventListener('keyup', onKeyUp);
  target.addEventListener('blur', onBlur);

  return () => {
    target.removeEventListener('pointerdown', onPointerDown);
    target.removeEventListener('pointerup', onPointerUp);
    target.removeEventListener('pointercancel', onPointerUp);
    target.removeEventListener('keydown', onKeyDown);
    target.removeEventListener('keyup', onKeyUp);
    target.removeEventListener('blur', onBlur);
  };
}
```
