# music.menu

**Tipo:** música de fundo (loop) — telas de menu (Home, Perfil, Ninho, Loja, etc.).
**Mood:** calmo, acolhedor, levemente pré-histórico/aventura leve; não intrusivo.
**Andamento:** ~84 BPM. **Loop:** 20–40s sem costura audível.
**Instrumentação sugerida:** pads suaves + melodia simples (flauta/marimba).
**Mix:** teto baixo (não abafar SFX/voz); ver `MUSIC_CEILING`.
**Formato-alvo:** `.ogg` (fallback `.mp3`), mono ou estéreo leve.
**Placeholder atual:** sequência senoidal em `MUSIC_TRACKS.menu` (`src/services/audio/tracks.ts`).
